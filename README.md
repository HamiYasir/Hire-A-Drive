## Hire A Drive

 A simple Uber like web application developed with the intention of learning more about React with MERN Stack. This features a user and driver clients that allows users to hire a cab and drivers to find these request. Current implementation is in beta stage and only features location in Kerala.

 ## TECHNOLOGIES USED

 - [React.Js](https://react.dev/): Used to build the frontend.
 - [MaterialUI](https://mui.com/material-ui/): Design elements used to design the web app.
 - [Express.Js](https://expressjs.com/en/): Middleware used to interact between frontend and backend.
 - [MongoDB](https://www.mongodb.com/): NoSQL database used to store client details, trip requests, etc.
 - [Node.Js](https://nodejs.org/en): Framework used to buld this project.
 - [Cloudinary](https://cloudinary.com/): Cloud storage used to store images and handle images using links.
